# flask-auth-ex1

## Install

    (prerequired) mysql # for flask_mysqldb

    git clone https://github.com/evadelzz1/flask-auth-ex3.git

    cd ./flask-auth-ex3/

    python -m venv .venv && source .venv/bin/activate

    pip install -r requirements.txt

    python app.py
